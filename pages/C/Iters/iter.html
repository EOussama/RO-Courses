Structures itératives

Boucle while (TanQue)

Algorithmique C
TantQue <condition> faire
<action>
FTQ 		while (<condition>)
<action>;

● <action> peut être une action simple ou composée, dans ce dernier cas, ellse sera délimitée par: {et}.

Algorithmique C
i ← 0
s ← 0
TantQue i <= 100 faire
i ← i + 1
s ← s + i
FTQ 		i = 0
s = 0
while (i <= 10)
{
i = i + 1
s = s + i
}

Boucle de...while (Faire...TanQue)

Algorithmique C
Répeéter:
<Action>
Jusqu'à: <Condition> 		do
<Action>
while (non <Condition>)
Exemple:
Algorithmique C
i ← 0
Répeéter:
i ← i + 1
Jusqu'à: i = 10 		i = 0
do
i = i + 1;
while (i != 10)

Boucle for (pour)

Algorithmique C
Pour <var> allant de <val initiale> à <val finale > faire
<Action>
FinPour 		for([<var> = <var initial>]; [<var> <= <val finale>]; [<var> = <var> + <pas-variation>])
<Action>
Exemple:
Algorithmique C
s ← 0 Pour i allant de 1 à 10 faire
s ← s + i
Ecrire("i = ", i, " et s = ", s)

FinPour 		s = 0;
for(i = 1; i <= 10; i = i + 1)
{
s = s + i;
printf("i = %d et s = %d \n", i, s);
}
