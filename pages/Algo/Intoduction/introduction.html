Cours d'algorithme

Introduction

Algorithme

● En informatique, un algorithme est une suite d'instructions permettant d'arriver à un résultat donné.

● Un algorithme est écrit dans un langage humain (simple).

Structure générale d'un algorithme

Notion de variable

● Les données ainsi que les résultats des calculs intermédiaires ou finaux, sont rangés dans des cases-mémoires appelées variables.

● Une variable:

  • Un emplacement mémoire nommé.
  • Prenant un nombre indéfini de valeurs différentes au cours du déroulement de l'algorithme.

Déclaration des variables

● La partie déclarations consiste à énumérer toutes les variables dont on aura besoin au cours de l'algorithme.

● Syntaxe:
Variable nom_de_la_variable : type
● Exemple:
variables nombre, k : entier
nom : chaine
a, b : booléen

Types des variables

● Entier:
les nombres entiers positifs ou négatifs
● Réel:
les nombres réels
● Caractère:
caractères alphabétiques et numérique
● Chaine:
une chaine de caractère
● Booléen:
les expressions logiques (Vrai, Faux)

Les constantes

● Comme une variable, une constante est un emplacement mémoire, mais dont la valeur stockée ne sera jamais modifiée au cours du programme.

● Syntaxe:
Constante nom_de_la_constante = valeur
● Exemple:
Constante pi = 3.14

Les instructions

● Une instruction est une action que l'ordinateur est capable d'exécuter.

● Chaque langage de programmation fournit une liste des instructions qu'on peut utiliser.

● Les instructions de base sont:

  • ● L'instruction d'affectation:
    L'affectation permet d'affecter une valeur à une variable.
    Symbolisée en algorithmique par une flesh, elle précise le sens de l'affectation.
    ● Syntaxe:
    Variable ← expression
    ● Expression peut être soit:
    Constante.
    Identificateur (nom d'une variable).
    Expression arithmétique.
    Expression logique.

Les opérateurs arithmétiques et relationnels


Algorithme Calcul
Variables A, B, C, D : entier
Début
A ← 10
B ← 30
C ← A+B
D ← C*A
Fin

● L'instruction de sortie: L'instruction de sortie (d'écriture) permet d'afficher des informations à l'utilisateur à travers l'écran.

● Syntaxe:
Ecrire (expression)
Expression peut être un message, le contenu d'une variable...

Algorithme Affichage
Variables A : entier
Début
A ← 10

Cette instruction permettra d'afficher bonjour à l'écran.
Ecrire("bonjour")

Cette instruction permet d'afficher à l'écran la valeur de la variable A.
Ecrire(A)

ou bien
Ecrire("la valeur de A est: " , A)
Fin

● L'instruction d'entrée:

  • ● L'instruction d'entrée ou de lecture permet á l'utilisateur de saisir des données au clavier.
  • ● Cette instruction assigne (affecte) une valeur entrée au clavier dans une variable.
  • ● Syntaxe: Lire (variable)

Algorithme Input
Variables A : entier
Début
Cette instruction permet à l'utilisateur de saisir une valeur au clavier qui sera affectée à la variable A.
Lire(A)

Fin

Remarque:

Lorsque le programme rencontre cette instruction, l'exécution s'interrompt et attend que l'utilisateur tape une valeur. Cette valeur est rangée en mémoire dans la variable désignée.
