Cours d'algorithme

un processus ou un ensemble de règles à suivre dans les calculs ou autres opérations de résolution de problèmes, en particulier par un ordinateur.

Cours de C

C est un langage de programmation de haut niveau et polyvalent, idéal pour le développement de microprogrammes ou d'applications portables. Initialement destiné ´ l'écriture de logiciels système, C a été développé par Bell Ritchie chez Bell Labs pour le système d'exploitation Unix au début des années 1970.

Cours de C#

C# est un hybride de C et C++, c'est un langage de programmation Microsoft dèveloppé pour rivaliser avec le langage Java de Sun. C# est un langage de programmation orienté objet utilisé avec les services Web XML sur la plate-forme .NET et conçu pour améliorer la productivité dans le développement d'applications Web.

Cours de Java

Java est un langage de programmation informatique polyvalent qui est concurrent, basé sur les classes, orienté objet, et spécifiquement conçu pour avoir le moins de dépendances possible.

Cours de SQL Server

SQL Server est le systéme de gestion de base de données relationnelle (SGBDR) de Microsoft. C'est une base de données complète principalement conçue pour rivaliser avec les concurrents Oracle Database (DB) et MySQL. Comme tous les principaux RBDMS, SQL Server prend en charge ANSI SQL, le langage SQL standard.

Cours de HTML

Hypertext Markup Language (langage de balisage hypertexte), un système standardisé pour baliser les fichiers texte afin d'obtenir des effets de police, de couleur, de graphique et d'hyperlien sur les pages du World Wide Web.
